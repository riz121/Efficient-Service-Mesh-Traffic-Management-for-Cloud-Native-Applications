# Reproducibility Scripts

Each script loads a CSV file from `../data/` and regenerates one figure from the paper.

## Run
```bash
python plot_fig6_response_time.py
python plot_fig7_no_traffic_mgmt.py
python plot_fig8_mcg_enabled.py
python plot_fig9_traffic_load.py
python plot_fig10_retry_timeout.py
python plot_fig11_resource_usage.py
```
